# Android SDK with Gradle

## Included
* OpenJDK 8
* Gradle 5.4.1
* Android SDK (android-29)
* Android Build-tools (29.0.3)
* Android Support Libraries
* Git

## Build image

```bash
docker build -t izapelini/android-sdk-gradle-ci .
```

## Run docker

```bash
docker run --rm -it izapelini/android-sdk-gradle-ci bash
```
